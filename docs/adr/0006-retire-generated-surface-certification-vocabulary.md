# Retire generated-surface certification vocabulary

The old project vocabulary grew around proving a generated 365-TR surface, which no longer matches the maintained SDK architecture. We will not carry overlapping terms such as Generated API Surface, Spec Drift Review, TR Certification, Certified TR Coverage, Evidence Accountability, or Fixture Coverage Debt into the new repository unless a term expresses a distinct remaining concept; maintained-surface, tracker, metadata, support-state, change-scoped gate, and focused-evidence language replaces them.
